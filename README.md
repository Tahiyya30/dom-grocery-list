DOM Grocery List

Overview
This program demonstrates basic JavaScript DOM manipulation. It displays a grocery list and highlights the items that have been bought from the list.

How It Works
1.) A grocery list variable is defined as an array containing grocery items.

2.) A function is created that takes an array as an argument and displays all the items in the grocery list by adding them to an HTML unordered list.

3.) Another function is created to highlight items that have been bought. The function takes the bought items and the grocery list array as arguments. It checks if the list exists, and if it does, the function highlights the matching items.
